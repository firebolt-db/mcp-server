# [](#radians)RADIANS

Converts degrees to radians.

## [](#syntax)Syntax

```
RADIANS(<value>) 
```

## [](#parameters)Parameters

Parameter Description Supported input types `<value>` The value to be converted to radians `DOUBLE PRECISION`

## [](#return-type)Return Type

`DOUBLE PRECISION`
